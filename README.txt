== Description ==

A simple Subscribe via RSS reminder at the end of your comment form.

Sometimes the most simple additions to your blog are the absolute best - no need to get overly complicated, right?

With this WordPress Plugin you can add a simple reminder at the end of your comment form to get your commenters (qualified and interested commenters) to subscribe to your RSS Feed!

<a href="http://tentblogger.com/rss-reminder">Read more here</a> as well as learn more about a perfect companion plugin that you'll want to have as well!

== Installation ==

1. Upload the plugin to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Done! Update style.css in the plugin's CSS directory to customize the look.
